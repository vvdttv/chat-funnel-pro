## Navigation
4-tab bottom nav: Funis, Atividades, Indicadores, Config

## Multi-Funnel System
Each funnel is a separate pipeline with its own stages:
- **MCMV** — Novo Lead → Simulação Crédito → Visita → Proposta → Contrato Assinado
- **Alto Padrão** — Novo Lead → Qualificação → Visita → Negociação → Fechamento
- **Aluguel** — Novo Lead → Visita → Análise Documentos → Contrato
- **Inquilinos** — Ativo → Renovação → Rescisão

Chat/inbox merged into deal detail sheet (Conversa tab). No separate inbox tab.
