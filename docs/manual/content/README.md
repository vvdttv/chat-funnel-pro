## Como ler

Se você está **decidindo se compra**: leia o 00 e dá uma passada visual no 01 e 04 (são os mais densos).

Se você **acabou de comprar e é admin**: leia 01 inteiro, depois 07. Os demais quando tiver tempo (você precisa entender o que cada papel vê pra ajudar).

Se você é **corretor / correspondente / administrativo / seguradora / vistoriador**: leia só o seu capítulo. É o suficiente.

## Sobre os prints

Todos os prints deste manual foram **capturados do sistema rodando**, não recriados. Cada print tem anotações em vermelho (setas, círculos, números) marcando onde clicar e o que esperar.

Quando um print mostra dados como "Joaquim Silva" ou "Rua das Acácias, 142", são dados fictícios de demonstração — seu sistema vai mostrar os dados reais da sua imobiliária.
